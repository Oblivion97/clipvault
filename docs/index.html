Features

Everything you need.
Nothing you don't.

ClipVault runs silently in the background and stays out of your way until you need it.

⌨️

Win+V Shortcut

Opens your clipboard history from anywhere, just like Windows 11. Registered automatically at install.

🔍

Instant Search

Start typing the moment the popup opens to filter your history in real time. No clicks needed.

🖼️

Image Previews

Copied images show live thumbnails in the list so you always know what you're pasting.

💾

Persistent History

Up to 200 items saved across reboots. Your history is always there when you need it.

🖥️

Wayland + X11

Full support for both display servers. Works on GNOME, KDE, Cinnamon, MATE, and more.

⚙️

Settings Page

Configure history limit, paste speed, image capture, privacy mode, and more — all in-app.

🔔

System Tray Icon

Lives quietly in your tray. Right-click to pause monitoring, clear history, or open settings.

🚀

Auto-starts

Registers itself to start silently at every login. Zero setup after install.

🐧

Cross-distro

Ubuntu, Fedora, Arch, Pop!_OS, Mint, openSUSE, Debian — one installer handles all.

Install

Up in 30 seconds.

The installer detects your distro and handles all dependencies automatically.

1

Debian / Ubuntu / Mint / Pop!_OS

sudo dpkg -i clipvault_..._all.deb
sudo apt-get install -f
2

Fedora

sudo dnf install clipvault-....noarch.rpm
3

openSUSE

sudo zypper install clipvault-....noarch.rpm
4

Arch / Manjaro / Any distro — Universal installer

wget https://github.com/Oblivion97/clipvault/releases/latest/download/clipvault-1.6.0..zip
unzip clipvault-....zip && cd clipvault-...
chmod +x install.sh && ./install.sh
